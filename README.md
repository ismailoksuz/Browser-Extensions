# Browser Extensions

This repository is a collection of various browser extensions developed to enhance privacy, security, and user experience. Each project is maintained within its own dedicated directory.

## Project Structure

Each extension is located in its own folder containing its specific manifest, assets, and scripts. To install an extension, load the corresponding folder into your browser's developer mode.

## Testing Laboratory

The `/test-files` directory contains specialized environments to verify the capabilities of the protocols:

* **`quantum_click_test.html`**: A target practice interface for **Quantum Click**. Features a haptic click counter to test independent cursor precision and CPS (Clicks Per Second) frequency.
* **`ghost_copy_test.html`**: A high-security simulation environment used to test the **Neural Link: Ghost Copy** bypass. It features locked right-click, disabled text selection, and copy-protection.

## Installation Guide

To use these extensions in a Chromium-based browser (Chrome, Brave, Edge, etc.):

1. Clone this repository to your local machine.
2. Open your browser and navigate to the extensions management page (`chrome://extensions`).
3. Enable "Developer mode" in the top right corner.
4. Click "Load unpacked" and select the directory of the extension you wish to install.
